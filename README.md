journalApp
----------------
A simple Spring Boot–based journal application built while learning Spring Boot.

Purpose of This Repo
----------------------
This project is not a production app. 
It’s a learning project where I explore:
    How to build REST APIs in Spring Boot
    How controllers, services, and repositories work
    How to structure a CRUD application
    How to connect Spring Boot with a database
    Basic request handling and JSON responses
    Experimentation with Spring Boot features

Features Implemented
-----------------------
Create a journal entry
View all entries
Update an entry
Delete an entry
Basic validation
database persistence

Tech Stack
-----------
Spring Boot
Java
Spring Web
MongoDB
Maven
